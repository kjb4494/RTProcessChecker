# RTProcessChecker
## 실시간 프로세스 검사 응용프로그램 (Real Time Process Checker) 
### 1. 기능
    1. DLL Injection 감지 
    2. 바이러스 및 악성 프로그램 감지 
    3. 브라우저가 접근하는 Domain 신뢰도 체크
### 2. 개발 환경
    Windows 10
### 3. 개발 도구
    1. PyCharm (Python 3.6)
    2. VirusTotal API
    3. Google Safe Browser API
### 4. 필요 모듈 설치
    pip install requests
    pip install psutil
### 5. 참고 자료
    1. 리버싱 핵심원리
    2. 점프 투 파이썬
    3. CrowdInspect.exe
